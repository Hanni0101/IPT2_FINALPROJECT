# IPT2_FINALPROJECT - Sizzle App

## Overview

A final project for our IPT2 Subject. This system has a public view for those without an account. Users can register and login, but the add to cart/order functionality is not yet operational. They can only view the menu for now. 

The system features an admin account for checking user count and menu items, with the ability to add new users and menu items to the system.

## Features

### Public Access
- Browse menu without account
- View restaurant information
- User registration and login

### User Features
- User authentication and login
- Menu browsing
- Account management

### Admin Features
- User management (view and add users)
- Menu management (view and add menu items)
- Dashboard for monitoring system activity
